# 软件工程期末复习资料汇总

## 仓库介绍

本仓库用于收集和整理软件工程普通专业的所有期末复习资料，供同专业同学参考学习。我们希望通过这个平台，帮助同学们共享学习资源，共同进步。

## 贡献指南

欢迎更多同学贡献自己的复习资料！如果您有好的资料想要分享，可以通过以下方式贡献：
1. Fork 本仓库
2. 添加您的资料到对应目录
3. 提交 Pull Request

贡献的资料将帮助更多同学，感谢您的支持！

## 内容结构

内容以学期/课程名/的文件夹进行组织，方便同学们按学期和课程查找所需资料。
