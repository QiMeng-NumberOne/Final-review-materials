# 软件工程学习资源汇总

## 仓库介绍

本仓库用于收集和整理软件工程普通专业的学习资源，包括期末复习资料、编程练习资源、技术教程等，供同专业同学参考学习。我们希望通过这个平台，帮助同学们共享学习资源，共同进步，打造一个全面的软件工程学习资源库。

## 贡献指南

欢迎更多同学贡献自己的学习资料！如果您有好的资料想要分享，可以通过以下方式贡献：
1. Fork 本仓库
2. 添加您的资料到对应目录
3. 提交 Pull Request

贡献的资料将帮助更多同学，感谢您的支持！

## 内容结构

内容按功能模块进行组织，方便同学们按需查找所需资源：
- 学期课程资料：按学期/课程名/的文件夹进行组织
- 技术教程：包含各类技术工具和技能的学习资料
- 编程练习资源：提供算法、编程题等练习资料
- 其他学习资源：包含学业帮扶、考试管理、培养方案等资料
