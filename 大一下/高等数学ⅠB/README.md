# 高等数学ⅠB

## 课程难度

本课程整体难度不大，是高等数学ⅠA的延续，主要涵盖多元函数微积分、级数、微分方程等内容。

## 学习建议

1. **重视课本习题**：认真完成课本中的课后习题，这是掌握知识点的基础。
2. **做好平时作业**：资料中的平时作业是对课堂内容的很好巩固，确保能够全部正确完成。
3. **理解基本概念**：重点理解多元函数的极限、导数、积分的基本概念和几何意义，掌握级数收敛判别法和微分方程的求解方法。
4. **多做练习**：通过大量练习加深对知识点的理解和应用能力，尤其是在多元积分和微分方程部分。
5. **复习课件和往年试题**：期末复习时，结合课件系统回顾所有知识点，并做一做资料中的往年试题，熟悉考试题型。

## 成绩预期

只要按照上述建议学习，认真完成课后习题和平时作业，取得90+的成绩是很容易的。
